Fancy WordPres XMAS-Calendar with animated doors.

== Description ==

WordPress module to build your own fancy XMAS-Calendar.
Its fully compatible with WordPress and GravityForms styles.

It's a WordPress-Custom-PostType.

The calendar endpoints are:

Calendar Page: www.YOURSITE.COM/xmas-calendar/
Thank you page: www.YOURSITE.COM/xmas-calendar/?lmt-xmas-calendar-thankyou=true
